# yuyu
已经用node完成了大文件上传下载，用java完成了入库操作，大文件上传接口/upload，下载接口/download，入库接口/api/pack
入库格式为
<pre>
{
"dass":0,
"date":1553307200,
"title":"test",
"url":"xxxxx"
}
</pre>
下载格式为
/download:identifier
<br>identifier就是点提交按钮后会跳出来显示的那个。
<br>另外，有些前端样式和脚本并不符合VUE的规范，希望大家改一改
